# Shell & Prompt Configurations

This directory contains configuration files for various shell environments and the prompt engine.

## Contents
* **[config.fish](file:///Users/abdurrofi/Documents/mrofisr/shell/config.fish)**: Settings, aliases, paths, and configurations for the Fish shell.
* **[profile.ps1](file:///Users/abdurrofi/Documents/mrofisr/shell/profile.ps1)**: Profile configuration for PowerShell, including modules, themes, and shell aliases.
* **[bashrc](file:///Users/abdurrofi/Documents/mrofisr/shell/bashrc)**: Configuration file for the Bash shell.
* **[oh-my-posh.json](file:///Users/abdurrofi/Documents/mrofisr/shell/oh-my-posh.json)**: Exported active theme and schema configuration for the Oh-My-Posh prompt engine.
